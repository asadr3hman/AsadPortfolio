Project Overview

Portfolio Project is a Node.js-based backend server designed to host and manage portfolio data for developers. This project enables users to register, log in, and securely manage their personal portfolio, including sections for projects, skills, accomplishments, education, experience, testimonials, and certifications. It provides a comprehensive solution for developers to showcase their work and track their career growth efficiently.

Features

  • User Registration and Login: Secure user authentication and authorization.
  • Portfolio Management: Create and manage portfolio entries such as projects, skills, and achievements.
  • RESTful API: A robust API for all functionalities, ensuring seamless data handling.
  • Security Features: Includes rate limiting, XSS protection, and CORS handling to ensure data security.

API Endpoints

  • POST /api/v1/auth/register: Register a new user.
  • POST /api/v1/auth/login: Log in an existing user.
  • POST /api/v1/projects: Create a new project entry.
  • GET /api/v1/projects: Retrieve all projects.
  • GET /api/v1/projects/:id: Get a specific project by ID.
  • PUT /api/v1/projects/:id: Update a project entry.
  • DELETE /api/v1/projects/:id: Delete a project entry.
  • POST /api/v1/skills: Add a new skill.
  • GET /api/v1/skills: Get all skills.
  • POST /api/v1/accomplishments: Add an accomplishment.
  • GET /api/v1/accomplishments: Get all accomplishments.
